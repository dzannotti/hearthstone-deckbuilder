# hearthstone-deckbuilder

A deck builder web application for Blizzard's Hearthstone. Built with AngularJS.

## Demo
[Hearthstone Deck Builder](http://deckbuilder.jumpcat.de)

## Installation

Serve ``dist/`` from your web server.

## Development

```bash
$ bower install
$ npm install
$ grunt
```
